
# Tourify

Tourify es una aplicación móvil diseñada para optimizar la exploración turística de ciudades, lugares y eventos. La aplicación se centra en la experiencia del usuario, gestionando un ecosistema completo de usuarios, ciudades, lugares de interés, eventos, reseñas, favoritos e interacciones mediante notificaciones.


## Estructura del repositorio

- `frontend/` — App cliente (Expo, React Native)
- `backend/` — API y lógica del servidor (Laravel)
